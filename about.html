How to use

Here a few instructions are given about the use of each page.

Home

The Home page contains a few key informations about the spendings and earnings stored in the application.

List

The List page, as the nome implies, is where all spendings and earnings as well as their related information are viewed and edited.

The formats expected for each information are the followings:

  • Name: no constraints;
  • Date: dd/mm/aaaa;
  • Value: numbers where spendings should be negative and earnings positive.

This application does not fix information inputted incorrectly, so please keep it in mind. Another impportant information to keep in mind is the fact that all modifications should be saved otherwise they will disappear by the next time the page is loaded

Analysis

This page will plot all spendings and earnings over the months.
